# DramaPix Starter for Render + Supabase

Este pacote é uma base inicial para hospedar o DramaPix.

## Estrutura
- `server.js` -> servidor Express para Render
- `package.json` -> dependências e scripts
- `.env.example` -> variáveis de ambiente
- `public/index.html` -> interface inicial do DramaPix
- `public/app.js` -> conexão com Supabase e ações básicas
- `public/styles.css` -> visual inicial

## 1) Instalar localmente
```bash
npm install
npm start
```

## 2) Variáveis de ambiente no Render
Crie estas variáveis:
- `SUPABASE_URL`
- `SUPABASE_ANON_KEY`
- `PORT` (opcional; Render já fornece)

## 3) Deploy no Render
- Suba estes arquivos para um repositório GitHub
- No Render, crie um novo **Web Service**
- Conecte o repositório
- Build command: `npm install`
- Start command: `npm start`

## 4) O que esta base já faz
- abre o app DramaPix
- conecta com Supabase
- permite cadastro/login básico
- mostra estrutura de abas principais
